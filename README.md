## гҖ” WHAT IS PRIVGUARD? гҖ•

**PrivGuard** is a sleek, cyberpunk-themed Android encryption app that lets you lock and unlock any text using **AES-128-CBC** вҖ” one of the most widely trusted encryption standards in the world. No accounts. No cloud. No tracking. Just pure, local encryption on your device.

Built entirely with **Kotlin** and **Jetpack Compose**, it's fast, minimal, and ruthlessly clean.

## гҖ” ARCHITECTURE гҖ•

PrivGuard follows **MVVM (Model-View-ViewModel)** вҖ” clean separation of concerns:

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ     state      в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ     logic     в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ  HomeScreen в”Ӯ в—„в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв–ә в”Ӯ  CryptoViewModel в”Ӯ в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв–ә в”Ӯ EncryptionManager в”Ӯ
в”Ӯ  (Compose)  в”Ӯ   user events  в”Ӯ   (ViewModel)    в”Ӯ               в”Ӯ  (AES-128/CBC)    в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ                в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ               в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

- **HomeScreen** вҶ’ Pure UI, observes ViewModel state
- **CryptoViewModel** вҶ’ Holds input/output/mode state, handles clipboard
- **EncryptionManager** вҶ’ Stateless singleton, handles all crypto operations

## гҖ” HOW IT WORKS гҖ•

```
YOUR MESSAGE                          ENCRYPTED OUTPUT
в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ                          в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
"Hello World"  в”Җв”Җв–ә AES-128-CBC в”Җв”Җв–ә  "rBxK9mZ2+Lp...=="
                   (encrypt)

"rBxK9mZ2+Lp...=="  в”Җв”Җв–ә AES-128-CBC в”Җв”Җв–ә  "Hello World"
                         (decrypt)
```

**Algorithm:** AES (Advanced Encryption Standard)  
**Mode:** CBC (Cipher Block Chaining)  
**Padding:** PKCS5  
**Encoding:** Base64 (for safe text transport)
